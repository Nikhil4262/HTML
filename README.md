# HTML Practice Projects (Without CSS)

This repository contains my HTML practice files created without using CSS or JavaScript. The main purpose of this repository is to strengthen my understanding of HTML fundamentals and semantic structure.

## 📌 About

These files are part of my HTML learning journey. Every page is built using pure HTML to understand how web pages are structured before moving on to styling and scripting.

- Line Breaks (`<br>`)
- Horizontal Rules (`<hr>`)
- Comments

- Subscript (`<sub>`)

### Links & Navigation

- Email Links

### Images & Media

### Tables

### Forms

### Semantic HTML
- Header

### Other Elements
